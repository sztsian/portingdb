Package python-oslo-utils

This package is released with dual support for Python 2 and Python 3. To move things forward, you can start making dependent packages use the Python 3 version.


This package has Ambiguous Requires (or BuildRequires): it uses unversioned python, rather than python2 or python3, to refer to other packages. The Python version should always be given explicitly.

See python-oslo-utils in the Fedora Pagure, or go directly to the corresponding spec file.

The tests subpackage is still Python2-only.
